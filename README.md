# 🏦 Bank Account Management System

A web-based Bank Account Management System developed using Python and Flask that allows users to manage accounts, perform secure transactions, and track financial activities.

---

## 🚀 Features

- 👤 User registration and login
- 💸 Money transfer between accounts
- 💰 Deposit and withdrawal
- 📄 Mini statement and passbook
- 🔐 OTP and PIN verification
- 💳 Card verification
- 📊 Spending limits and statistics
- 🛑 Account lock/unlock
- 🔎 Search account details
- 📉 Loan management

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML / CSS
- SQLite Database
